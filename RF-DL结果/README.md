RF-DL Results

This directory contains output files generated by the RF-DL pipeline.

Files:
dl_model.pt: Trained deep learning model.
feature_norm.npz: Normalized feature data.
processed_pairs.csv: Processed protein pair data.
rf_classification_report.csv: Classification report of the random forest model.
rf_dl_auc_roc.png: ROC curve of the RF-DL model.
rf_dl_main_metrics.csv: Main evaluation metrics of the RF-DL model.
rf_dl_processed_pairs.csv: Test set predictions with RF scores.
top_proteins_rf_dl.txt: Top-ranked proteins based on RF-DL predictions.

Subdirectories:
go_enrichment: GO enrichment analysis results.
kegg_enrichment: KEGG pathway enrichment analysis results.

PPI-RF-DL

This repository contains code and results for Hybrid Random Forest–Deep Learning modeling for Protein–Protein Interaction Prediction .
The project includes data preprocessing, model training, evaluation, and
functional enrichment analysis.
See the RF-DL results directory for model outputs and analysis results.

The raw molecular dynamics simulation data used in this study are publicly available on Figshare: https://doi.org/10.6084/m9.figshare.31167793
